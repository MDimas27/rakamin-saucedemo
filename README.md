# SauceDemo Testing Project

This Maven project is designed for testing SauceDemo using Selenium and Cucumber.

## Project Structure

- **src/main/java**: Contains the main source code for the project.
- **src/test/java**: Contains test cases using JUnit and Cucumber.
- **pom.xml**: Maven project configuration file.

## Dependencies

- **JUnit 4.12**: Testing framework.
- **Selenium 4.14.1**: Web browser automation.
- **JUnit Vintage Engine 5.10.0**: Allows running JUnit 4 tests with JUnit 5.
- **WebDriverManager 5.5.3**: Simplifies the management of WebDriver binaries.
- **Cucumber 7.14.0**: Behavior Driven Development (BDD) tool for Java.
